# אתר מעקב אימונים – תכנית יישום

אתר HTML סטטי פשוט שמאפשר מעקב אחרי שני אימונים שבועיים, עם צ'קליסט לתרגילים וטיימר מנוחה מובנה.

---

## עיצוב כללי

- **שפה**: HTML + CSS + Vanilla JS (קובץ אחד או שלושה קבצים נפרדים)
- **מצב כהה** כברירת מחדל – נראה פרימיום ואתלטי
- **גופן**: `Inter` מ-Google Fonts
- **פלטת צבעים**: אפור כהה (#0f0f0f, #1a1a1a) עם אקסנט כחול-נאון (#00d4ff) או סגול (#a855f7)
- **אנימציות עדינות**: מעבר צבע בעת סימון תרגיל, pulse על הטיימר

---

## מבנה הדפים / סקציות

### 1. Header
- שם האתר (למשל "Training Tracker" / "מעקב אימונים")
- תאריך היום (עודכן אוטומטית)

### 2. בחירת אימון
- שני כפתורים / טאבים: **אימון 1** | **אימון 2**
- לחיצה על אחד מציגה את הצ'קליסט הרלוונטי

### 3. צ'קליסט האימון
כל תרגיל יוצג כשורה עם:
- ✅ **צ'קבוקס** לסימון שהתרגיל בוצע
- שם התרגיל + מספר סטים/חזרות
- כפתור **"מנוחה"** ▶ שמפעיל את הטיימר לאחר כל סט (אופציונלי)
- אפקט strikethrough + שינוי צבע כשמסמנים

### 4. טיימר מנוחה (Rest Timer)
- Widget צף (קבוע בפינה או מופיע בתחתית)
- ברירת מחדל: 60 שניות (ניתן לשנות ל-90 / 120 / custom)
- כפתורי: **Start / Pause / Reset**
- **Progress ring** (SVG עגול) שמתמלא תוך כדי
- צליל (Beep) או רטט כשהזמן נגמר (Web Audio API)

### 5. Progress Bar לאימון
- בר קטן בראש הצ'קליסט שמראה "X / Y תרגילים הושלמו"
- Confetti קטן או הנפשה כשמסיימים 100%

## נתונים (אימונים)

האימונים יוגדרו כ-JSON בתוך `app.js` וקל יהיה לערוך אותם:

```json
{
  "workout1": {
    "name": "אימון 1",
    "exercises": [
      { "name": "סקוואט", "sets": 4, "reps": "8-10", "rest": 90 },
      { "name": "לחיצת חזה", "sets": 4, "reps": "8-10", "rest": 90 }
    ]
  },
  "workout2": {
    "name": "אימון 2",
    "exercises": [...]
  }
}
```

> אם יש לך תכנית אימון קיימת שתרצה להכניס – תשלח ואני אייבא הכל אוטומטית.

## שמירת מצב (localStorage)

- הצ'קבוקסים נשמרים לפי תאריך – כך שמחר מתאפס אוטומטית
- האימון האחרון שנבחר נשמר

---

## שאלות פתוחות

> [!IMPORTANT]
> **יש לך תכנית אימון קיימת?**
> תוכל לשלוח את שמות התרגילים, הסטים והחזרות – אייבא אותם אוטומטית לקוד במקום placeholder.

> [!IMPORTANT]
> **שמות האימונים**: האם "אימון 1" ו"אימון 2" הם הכינויים הנכונים, או שיש שמות ספציפיים (למשל "Push" / "Pull", "ראש-כתפיים" / "גב-ביצפס")?

> [!NOTE]
> **טיימר**: כמה שניות מנוחה ברירת מחדל? (60 / 90 / 120?)

---

## תכנית אימות

- בדיקה שהצ'קליסט מסמן ומנקה כראוי
- בדיקה שהטיימר מתאפס ומציג נכון
- בדיקה שה-localStorage שומר ומשחזר את המצב
- בדיקה ויזואלית של העיצוב (dark mode, ניידים + מחשב)
